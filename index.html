Curso de Git & GitHub

¿Qué es Git?

Git es un sistema de control de versiones para el mantenimiento eficiente y confiable de archivos.

¿Qué es GitHub?

GitHub es una plataforma de desarrollo colaborativo de software para alojar proyectos utilizando el sistema de control de versiones Git.

¿Qué es GitLab?

GitLab es una plataforma de desarrollo colaborativo de software para alojar proyectos utilizando el sistema de control de versiones Git.
